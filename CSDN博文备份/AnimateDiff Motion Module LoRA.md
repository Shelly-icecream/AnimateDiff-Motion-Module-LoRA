基于时间模块微调实现慢动作生成（完整复现与踩坑记录）</p> <br><h2><a id="_5"></a>引言</h2> <br><p>近年来，基于扩散模型的视频生成方法发展迅速，其中 AnimateDiff 由于无需重新训练基础模型即可实现视频生成，受到了广泛关注。在实际复现过程中，我发现官方提供的 Motion Module 已经能够学习一般的时间连续性，但针对 特定动态风格（例如慢动作） 的系统化实践资料相对较少。</p> <br><p>目前社区中关于 LoRA 的应用大多集中在画面风格或角色外观，而针对时间维度（temporal dynamics）的 LoRA 微调经验较为分散，尤其是如何通过 Motion Module 引导模型生成稳定的慢动作效果，缺乏完整的工程实践记录。</p> <br><p>因此，本项目尝试在 AnimateDiff 框架下，对 Motion Module 进行 LoRA 微调，并结合数据筛选与时间动态标签设计，探索一种 无需修改模型结构即可实现慢动作生成的训练流程。本文将从环境配置、数据准备、代码修改到训练与推理，记录完整复现与踩坑过程，希望为后续进行视频时序控制研究的同学提供参考。</p> <br><h2><a id="_11"></a>项目目标说明</h2> <br><p>本文基于：<br /> 动画框架：AnimateDiff<br />
